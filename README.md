# Friendship Day Surprise (Quick)


Open `index.html` in your browser to view the fun Friendship Day page. It's a single static page—no server required.

Files:
- `index.html` — main page (now includes decorative photo slots)
- `styles.css` — styling and simple animations
- `script.js` — interactivity: roasts, sweet messages, confetti

To run: double-click `index.html` or open it in your browser.

Photos:
- Add up to 4 photos for the decorative background in the `images/` folder with these filenames:
	- `images/photo1.jpg`
	- `images/photo2.jpg`
	- `images/photo3.jpg`
	- `images/photo4.jpg`
- If you don't add photos, the layout still works; replace names with your photo files to personalize.

Want more personalization? Tell me Bauni's real name, inside jokes, or whether to change roast tone.
